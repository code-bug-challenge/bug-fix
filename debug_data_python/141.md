你题目的解答代码如下：

n = int(input())
nums = list(map(int,input().split()))
tri = [nums]
for i in range(2000):
    sub = [0]*n
    p = tri[-1]
    for j in range(n):
        sub[j] = (p[j]+p[(j+1)%n])%998244353
    tri.append(sub)
