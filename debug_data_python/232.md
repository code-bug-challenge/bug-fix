题目要求是营销额1000刀以下奖10%的营销额，100刀以上奖15%营销额，输入营销额，如果输入是正数，在运行完一次后不会退出代码，而是再次循环，直到输入负数时输出“输入错误”，并结束进程

```python
sales = float(input("Enter sales：$"))
bonus = 0  # bonus
while ("sales >= 0"):
    if sales <= 1000:
        bonus = sales * 0.1
        print('A bonus of $%s shall be paid' % bonus)
        break
    elif sales >= 1000:
        bonus = sales * 0.15
        print('A bonus of $%s shall be paid' % bonus)
        break
    else:
        print("Please check the input")


```

### 修改方案：
