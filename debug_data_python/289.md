### 修改方案：
有两个花括号，后面format括号里怎么只有一个值

```python
import math

id = 0

print("天才是由{:.0%}的灵感，加上{:.0%}的汗水。".format(0.01,0.99))

while True:
    lis = input("输入商品名 单价,按0 退出:").split()
    if "".join(lis) != "0":
        id += 1
        print("%06d\t"% id+"    ",end="")
        print("{:s},{:.2f}元".format(lis[0],float(lis[1])))
    else:
        break

```
