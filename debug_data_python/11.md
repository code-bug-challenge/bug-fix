### 修改方案：
搞明白了,其实就是字符串传参的事,我重构下程序,没问题了

```python
def ky(iuy,f2):
    q1='(?<='
    q2=')\s*\d*\.\d*'
    q=q1+iuy+q2
    p = re.compile(q)
    a=list(map(float,p.findall(f2)))
    if a!=[]:
        print(iuy,a)
```
